## Initial Setup
- Set keyboard to DVORAK and remove QWERTY
- Remote Tags
- Set trackpad behavior
- Touchbar Setup
- Touch Mar shows: Expanded Control Strip
- Remove Siri Button
- Set computer name
- Change Finder Prefs (New windows default to ~/, don't show External Disks etc on desktop)
